### Syncfusion Blazor QueryBuilder Component

This package contains the [Blazor QueryBuilder](https://www.syncfusion.com/blazor-components/blazor-query-builder?utm_source=nuget&utm_medium=listing&utm_campaign=blazor-query-builder-nuget) for Blazor application. The Blazor Query Builder is a rich and responsive UI for filtering large amounts of data by creating or editing conditions that can be combined with data visualization controls like DataGrid and Charts to view the filtered data. It outputs structured JSON filters that can be easily parsed to create SQL queries. It allows you to create conditions and group them using AND/OR logic.
